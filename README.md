# E-Commerce Sales Analysis 2024

**Tools:** Python · Pandas · Plotly  
**Dataset:** 500 orders, Italian e-commerce, Jan–Dec 2024

## Objective
Analyze sales performance across categories, regions and time to identify growth opportunities and revenue drivers.

## Key Findings
- **Electronics** accounts for **35% of total revenue** (€46.3k/€132.3k)
- **Nord Italia** generates **33% of revenue** — highest regional concentration
- Seasonal peak in **April and Q4** — opportunity for targeted promotions
- Average discount of **5.2%** — healthy margin position
- Top 10 customers represent **~18% of revenue** — manageable concentration

## Files
- `ecommerce_sales_2024.csv` — cleaned dataset
- `ecommerce_analysis.py` — full analysis script
- `charts/` — exported visualizations

## How to Run
```bash
pip install pandas plotly
python ecommerce_analysis.py
```
